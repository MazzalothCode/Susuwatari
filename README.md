# Cat a font

Cat a font is an experimental front-end project where a single English letter is
suggested by a drifting colony of cat-head particles.

## Stack

- Vite
- React
- TypeScript
- PixiJS
- opentype.js

## Current scope

- Single English glyph input
- Several selectable type styles
- GPU-driven particle stage
- Video export planned as a later milestone

## Setup

Install dependencies with your preferred package manager, then run:

```bash
pnpm dev
```

or:

```bash
npm install
npm run dev
```
